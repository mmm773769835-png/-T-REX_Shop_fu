## 📊 **معلومات إضافية:**

### الملفات المتعلقة بالمدير:

| الملف | الوصف |
|-------|-------|
| `server/.env` | بيانات الاعتماد المشفرة |
| `server/index.js` | كود المصادقة (السطر 233) |
| `server/set-admin-password.js` | سكريبت إنشاء كلمة المرور |
| `src/screens/AdminLoginScreen.tsx` | واجهة تسجيل الدخول |

### الكود المسؤول عن التحقق:

**server/index.js (السطر 225-245):**
```javascript
app.post('/api/auth/admin-login', 
  validate([
    body('email').isEmail().withMessage('البريد الإلكتروني غير صحيح'),
    body('password').notEmpty().withMessage('كلمة المرور مطلوبة')
  ]),
  async (req, res) => {
    const { email, password } = req.body;
    
    const adminEmail = process.env.ADMIN_EMAIL || 'admin@t-rex-shop.com';
    const adminHash = process.env.ADMIN_HASH;
    
    if (email !== adminEmail) {
      return res.status(401).json({
        error: 'unauthorized',
        message: 'البريد الإلكتروني أو كلمة المرور غير صحيحة'
      });
    }
    
    const ok = await bcrypt.compare(password, adminHash);
    if (!ok) {
      return res.status(401).json({
        error: 'unauthorized',
        message: 'البريد الإلكتروني أو كلمة المرور غير صحيحة'
      });
    }
    
    // Generate JWT token...
  }
);
```

**الحل:**
```bash

node index.js
```

**الحل:**
```bash

```
