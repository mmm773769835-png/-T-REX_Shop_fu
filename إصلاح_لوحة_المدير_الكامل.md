## 📋 **التغييرات المطلوبة:**

### 1️⃣ **تحديث IP الشبكة**

**الملف:** `config/api.config.js`

```javascript
// القديم (خطأ)
const LOCAL_IP = '10.152.40.79';

// الجديد (صحيح)
const LOCAL_IP = '172.20.44.16';
```

**كيفية معرفة IP الصحيح:**
```powershell
ipconfig | findstr /i "IPv4"
```

### 2️⃣ **تحديث CORS في السيرفر**

**الملف:** `server/.env`

```env
# القديم
ALLOWED_ORIGINS=http://localhost:19006,http://localhost:19002,http://localhost:8081

# الجديد (يشمل Expo Go على الهاتف)
ALLOWED_ORIGINS=http://localhost:19006,http://localhost:19002,http://localhost:8081,exp://172.20.44.16:19000,exp://10.152.40.79:19000
```

### 3️⃣ **تعطيل CSRF للتطبيق**

**الملف:** `server/index.js`

```javascript
// تم تعطيل CSRF للتطبيق المحمول
// const csrfProtection = csrf({ cookie: { ... } });
// app.use(csrfProtection);
```

**السبب:**
- CSRF للويب فقط
- التطبيق المحمول لا يحتاج CSRF
- كان يسبب خطأ `"misconfigured csrf"`

### 4️⃣ **إصلاح Twilio**

**الملف:** `server/index.js` (السطر 102-114)

```javascript
let twilioClient = null;
try {
  if (process.env.TWILIO_SID && process.env.TWILIO_AUTH_TOKEN && 
      process.env.TWILIO_SID.startsWith('AC')) {
    twilioClient = Twilio(process.env.TWILIO_SID, process.env.TWILIO_AUTH_TOKEN);
    console.log('✅ Twilio client initialized');
  } else {
    console.log('⚠️  Twilio not configured - OTP will be logged to console');
  }
} catch (error) {
  console.log('⚠️  Twilio not available - OTP will be logged to console');
}
```

**النتيجة:**
- ✅ لا خطأ عند بدء التشغيل
- ✅ OTP يظهر في Console إذا لم يكن Twilio مُعد
- ✅ يمكن استخدام Twilio لاحقاً بسهولة
