# 🔗 كيفية استخدام سيرفر موجود في مجلد آخر

## 📍 الخطوات

### 1️⃣ حدّث ملف الإعدادات

افتح ملف `config/api.config.js` وحدّث القيم التالية:

#### إذا كان السيرفر على الإنترنت (Production):

```javascript
const USE_LOCAL_SERVER = false;
const PRODUCTION_API_URL = "https://your-server.com"; // عنوان السيرفر الفعلي
```

#### إذا كان السيرفر محلياً (على جهاز آخر في نفس الشبكة):

```javascript
const USE_LOCAL_SERVER = true;
const USE_IP = true;
const LOCAL_IP = '192.168.1.100'; // IP address الجهاز الذي يعمل عليه السيرفر
const PORT = 3000; // المنفذ (غير إذا كان مختلفاً)
```

#### إذا كان السيرفر على نفس الجهاز (localhost):

```javascript
const USE_LOCAL_SERVER = true;
const USE_IP = false;
const USE_LOCALHOST = true;
const PORT = 3000;
```

### 2️⃣ تحقق من عنوان السيرفر

بعد التحديث، ستظهر رسالة في console عند تشغيل التطبيق:
```
🔗 API Server URL: http://...
```

### 3️⃣ تأكد من أن السيرفر يعمل

- إذا كان السيرفر محلياً: تأكد أنه يعمل على الجهاز المحدد
- اختبر الاتصال بفتح: `http://YOUR_SERVER/api/auth/check` في المتصفح

### 4️⃣ جرّب التطبيق

الآن يمكنك استخدام OTP مع السيرفر الخاص بك!

## 📝 أمثلة

### مثال 1: سيرفر على الإنترنت
```javascript
const USE_LOCAL_SERVER = false;
const PRODUCTION_API_URL = "https://api.myshop.com";
```

### مثال 2: سيرفر على جهاز آخر في نفس الشبكة
```javascript
const USE_LOCAL_SERVER = true;
const USE_IP = true;
const LOCAL_IP = '192.168.1.50';
const PORT = 8080; // إذا كان المنفذ مختلف
```

### مثال 3: سيرفر على نفس الجهاز (localhost)
```javascript
const USE_LOCAL_SERVER = true;
const USE_IP = false;
const USE_LOCALHOST = true;
const PORT = 3000;
```

## ⚠️ ملاحظات مهمة

1. **للهاتف الحقيقي أو Android Emulator:**
   - استخدم IP address وليس localhost
   - تأكد أن الهاتف والكمبيوتر على نفس الشبكة WiFi

2. **لـ iOS Simulator:**
   - يمكن استخدام localhost

3. **للسيرفر الخارجي:**
   - تأكد أن السيرفر يسمح بـ CORS
   - استخدم HTTPS في الإنتاج

## 🔍 استكشاف الأخطاء

### خطأ "Network Error"
- ✅ تأكد أن السيرفر يعمل
- ✅ تحقق من العنوان في `config/api.config.js`
- ✅ اختبر الاتصال في المتصفح

### خطأ "CORS"
- ✅ تأكد أن السيرفر يسمح بـ CORS
- ✅ في السيرفر، أضف: `app.use(cors());`
