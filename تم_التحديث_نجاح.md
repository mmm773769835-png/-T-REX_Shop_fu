# ✅ تم تحديث الإعدادات بنجاح!

## 🎉 ما تم عمله

تم تحديث ملف `config/api.config.js` لاستخدام IP address جهازك:
- **IP Address:** `10.56.255.204`
- **Port:** `3000`

## 📝 الخطوات التالية

### 1️⃣ تأكد أن السيرفر يعمل

إذا كان لديك سيرفر في مجلد آخر:
- افتح Terminal في مجلد السيرفر
- شغّل السيرفر (مثلاً: `npm start` أو `node index.js`)
- تأكد أنه يعمل على المنفذ **3000**

### 2️⃣ إذا كان المنفذ مختلفاً

إذا كان السيرفر يعمل على منفذ آخر (مثلاً: 8080, 5000):
- افتح `config/api.config.js`
- غيّر: `const PORT = 3000;` إلى المنفذ الصحيح

### 3️⃣ أعد تحميل التطبيق

بعد التأكد من السيرفر:
1. اضغط `r` في Terminal لتحديث التطبيق
2. أو أعد تشغيل التطبيق

### 4️⃣ تحقق من Console

عند تشغيل التطبيق، ستظهر رسالة:
```
🔗 API Server URL: http://10.56.255.204:3000
```

تأكد أن العنوان صحيح!

### 5️⃣ اختبر الاتصال

افتح المتصفح واذهب إلى:
```
http://10.56.255.204:3000/api/auth/check
```

يجب أن ترى: `{"ok":false}` (هذا يعني السيرفر يعمل ✅)

## ⚠️ ملاحظات مهمة

1. **السيرفر يجب أن يعمل** قبل تجربة OTP
2. **الهاتف والكمبيوتر** يجب أن يكونا على نفس شبكة WiFi
3. **Firewall** قد يحتاج لإذن للسماح بالاتصالات

## 🔍 إذا كان السيرفر على منفذ آخر

إذا كان السيرفر يعمل على منفذ مختلف (مثلاً 8080):
- حدّث `config/api.config.js`:
  ```javascript
  const PORT = 8080; // المنفذ الصحيح
  ```
