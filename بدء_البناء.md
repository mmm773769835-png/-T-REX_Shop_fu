## 🚀 خيارات البناء

### 1. بناء Preview (معاينة) - موصى به للبدء
```bash
eas build --platform android --profile preview
```
- **النوع:** APK
- **الاستخدام:** اختبار على أجهزة حقيقية
- **المدة:** حوالي 10-20 دقيقة
- **التكلفة:** قد يتطلب رصيد EAS (حسب الخطة)

### 2. بناء Production (إنتاج)
```bash
eas build --platform android --profile production
```
- **النوع:** App Bundle (AAB)
- **الاستخدام:** النشر على Google Play
- **المدة:** حوالي 10-20 دقيقة
- **التكلفة:** قد يتطلب رصيد EAS (حسب الخطة)

### 3. بناء Development (تطوير)
```bash
eas build --platform android --profile development
```
- **النوع:** APK مع Development Client
- **الاستخدام:** التطوير والاختبار
- **المدة:** حوالي 10-20 دقيقة

## ⚠️ ملاحظات مهمة

### 1. حدود البناء المجانية
- حساب EAS المجاني له حد شهري لعمليات البناء
- إذا وصلت للحد الأقصى، قد تحتاج للانتظار أو ترقية الحساب

### 2. التوقيع (Signing)
- للبناء الأول، سيتم إنشاء keystore تلقائياً بواسطة EAS
- **مهم جداً:** احفظ بيانات الاعتماد (credentials) الخاصة بك

### 3. المدة
- البناء يستغرق عادة 10-20 دقيقة
- يمكنك متابعة التقدم على: https://expo.dev/accounts/trexshop/builds
