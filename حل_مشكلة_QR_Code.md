# 🔧 حل مشكلة عدم استجابة QR Code - Expo Go

## ❌ **المشكلة:**
عند مسح رمز QR لا يحدث شيء.

---

## ✅ **الحلول السريعة:**

### 1️⃣ **تأكد من تطبيق Expo Go:**

#### Android:
```
✅ نزّل من Google Play
✅ ابحث عن "Expo Go"
✅ تأكد من أنه آخر إصدار
✅ افتح التطبيق
```

#### iOS:
```
✅ نزّل من App Store  
✅ ابحث عن "Expo Go"
✅ تأكد من أنه آخر إصدار
✅ افتح التطبيق
```

**رابط التحميل:**
- Android: https://play.google.com/store/apps/details?id=host.exp.exponent
- iOS: https://apps.apple.com/app/expo-go/id982107779

---

### 2️⃣ **طريقة المسح الصحيحة:**

```
الخطوات:
1. افتح تطبيق Expo Go
2. انقر على "Scan QR Code" أو المسح الضوئي
3. وجّه الكاميرا نحو رمز QR
4. اقترب ببطء حتى يركز الكود
5. انتظر حتى يظهر خط أخضر
6. التطبيق سيفتح تلقائياً
```

**نصائح:**
- ✅ استخدم إضاءة جيدة
- ✅ ثبت الهاتف جيداً
- ✅ لا تقترب كثيراً ولا تبتعد كثيراً
- ✅ نظّف عدسة الكاميرا

---

### 3️⃣ **التحقق من اتصال الشبكة:**

```bash
# تأكد من أن الهاتف والكمبيوتر على نفس الشبكة WiFi

# على الكمبيوتر:
ipconfig | findstr /i "IPv4"

# يجب أن يكون IP مثل: 172.20.x.x أو 192.168.x.x
```

**إذا لم يكونوا على نفس الشبكة:**
- وصّل الهاتف بنفس راوتر الكمبيوتر
- أو استخدم Hotspot من الكمبيوتر للهاتف

---

### 4️⃣ **طرق بديلة للفتح:**

#### الطريقة أ: من المتصفح
```
افتح المتصفح على الهاتف:
http://172.20.44.16:8081
```

#### الطريقة ب: يدوياً في Expo Go
```
1. افتح Expo Go
2. اختر "Enter address manually"
3. اكتب: exp://172.20.44.16:8081
4. اضغط Connect
```

#### الطريقة ج: من الكمبيوتر
```
1. في التيرمينال، اضغط حرف 'w'
2. سيفتح في المتصفح
3. انسخ الرابط وأرسله للهاتف
```

---

### 5️⃣ **إعادة تشغيل التطبيق:**

```bash
# في شاشة التيرمينال:
اضغط Ctrl+C لإيقاف التطبيق

# ثم أعد التشغيل:
npx expo start -c
```

**النتيجة:**
```
› Metro waiting on exp+trex-shop://...
› Scan the QR code above
```

---

### 6️⃣ **مسح Cache وإعادة تعيين:**

```bash
# أوقف التطبيق (Ctrl+C)

# امسح الكاش:
npx expo start -c --clear

# سيعيد بناء الكاش من جديد (قد يأخذ دقيقة)
```

---

### 7️⃣ **التحقق من الجدار الناري (Firewall):**

#### Windows Firewall:

```powershell
# شغّل PowerShell كمسؤول (Run as Administrator)

# أضف قاعدة للسماح بـ Node.js:
New-NetFirewallRule -DisplayName "Node.js Expo" -Direction Inbound -Program "C:\Program Files\nodejs\node.exe" -Action Allow

# أو عطّل الجدار مؤقتاً للتجربة:
Set-NetFirewallProfile -Profile Domain,Public,Private -Enabled False
```

**⚠️ تحذير:** عطّل الجدار فقط للتجربة وأعد تفعيله بعد ذلك!

---

### 8️⃣ **منافذ الاتصال:**

```
المنفذ المطلوب: 8081 (للـ Metro Bundler)
منفذ السيرفر: 4001 (للـ API)

تحقق من المنافذ:
netstat -ano | findstr :8081
netstat -ano | findstr :4001
```

**إذا كانت المنافذ مغلقة:**
```bash
# أعد تشغيل التطبيق:
npx expo start -c
```

## 💡 **حلول خاصة:**

### المشكلة 1: "QR code not recognized"

**الحل:**
```
1. قرّب الهاتف أكثر (20-30 سم)
2. ثبّت الهاتف لمدة 3-5 ثواني
3. غيّر الزاوية قليلاً
4. استخدم إضاءة أفضل
```

---

### المشكلة 2: "Network error" أو "Couldn't connect"

**الحل:**
```bash
# 1. تحقق من IP الصحيح:
ipconfig | findstr /i "IPv4"

# 2. تأكد من نفس الشبكة:
WiFi Name على الهاتف = WiFi Name على الكمبيوتر

# 3. استخدم Tunnel mode:
في التيرمينال، اضغط 't'
```

---

### المشكلة 3: "App crashes on scan"

**الحل:**
```
1. أعد تشغيل الهاتف
2. امسح Cache لـ Expo Go
3. أعد تثبيت Expo Go
4. جرّب هاتف آخر
```

---

### المشكلة 4: "Loading... then nothing"

**الحل:**
```bash
# 1. أوقف التطبيق وأعد تشغيله:
Ctrl+C
npx expo start -c

# 2. امسح الكاش:
npx expo start -c --clear

# 3. تحقق من السيرفر:
cd server
node index.js
```
