## 🚨 **استكشاف الأخطاء:**

### المشكلة 1: "Build failed"
**الحل:**
```bash
# تحقق من logs
https://expo.dev/accounts/trexshop/projects/trex-shop/builds/[BUILD_ID]

# أعد المحاولة
npx eas build --platform android --profile preview --clear-cache
```

### المشكلة 2: "Timeout"
**الحل:**
- انتظر 5 دقائق ثم حاول مرة أخرى
- الخوادم قد تكون مشغولة

### المشكلة 3: "Invalid credentials"
**الحل:**
```bash
# حدّث بيانات الاعتماد
eas credentials

# ثم أعد البناء
npx eas build --platform android
```
