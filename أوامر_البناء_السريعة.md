# أوامر البناء السريعة - T-REX Shop

## 🚀 الأوامر الأساسية

### بناء Android

```bash
# بناء تطوير (Development)
eas build --platform android --profile development

# بناء معاينة (Preview) - APK للاختبار
eas build --platform android --profile preview

# بناء إنتاج (Production) - AAB للنشر
eas build --platform android --profile production
```

### بناء iOS

```bash
# بناء تطوير
eas build --platform ios --profile development

# بناء معاينة
eas build --platform ios --profile preview

# بناء إنتاج
eas build --platform ios --profile production
```

### بناء كلا المنصتين

```bash
# بناء إنتاج لكلاهما
eas build --platform all --profile production
```

## 🔧 البناء المحلي

### Android

```bash
# بناء محلي (يتطلب Android Studio)
npx expo run:android

# بناء APK محلي
cd android && ./gradlew assembleRelease
```

### iOS

```bash
# بناء محلي (يتطلب macOS و Xcode)
npx expo run:ios
```

## 📦 النشر على المتاجر

### Google Play Store

```bash
# بناء ونشر تلقائي
eas build --platform android --profile production --auto-submit

# أو نشر يدوي بعد البناء
eas submit --platform android
```

### Apple App Store

```bash
# بناء ونشر تلقائي
eas build --platform ios --profile production --auto-submit

# أو نشر يدوي بعد البناء
eas submit --platform ios
```

## 📝 تحديث الإصدار

### التحديث التلقائي

تم تفعيل `autoIncrement` في `eas.json` للإنتاج، لذا سيتم تحديث `versionCode` تلقائياً.

### التحديث اليدوي

في `app.json`:
```json
{
  "version": "2.0.1",  // تحديث رقم الإصدار المرئي
  "android": {
    "versionCode": 3  // زيادة رقم الإصدار الداخلي
  }
}
```

## ⚡ نصائح سريعة

1. **للبناء السريع:** استخدم `--local` للبناء المحلي
2. **للتحديثات السريعة:** استخدم `development` profile
3. **للنشر:** استخدم `production` profile
4. **للاختبار:** استخدم `preview` profile
