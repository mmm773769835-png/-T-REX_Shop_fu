## 📱 **طرق التثبيت:**

### الطريقة 1: التنزيل المباشر (الأسهل)

1. **من الكمبيوتر:**
   ```bash
   # افتح الرابط من Expo
   https://expo.dev/.../builds/[BUILD_ID]
   
   # انقر "Download APK"
   ```

2. **نقل APK للهاتف:**
   - USB Cable
   - Google Drive / Dropbox
   - WiFi Transfer app

3. **التثبيت:**
   - افتح APK على الهاتف
   - وافق على "Unknown sources"
   - انتظر التثبيت

### الطريقة 2: Expo Go (للtesting السريع)

1. **نزّل Expo Go:**
   - [Android Play Store](https://play.google.com/store/apps/details?id=host.exp.exponent)
   - [iOS App Store](https://apps.apple.com/app/expo-go/id982107779)

2. **امسح QR Code:**
   - من Expo Dashboard
   - أو من التيرمينال

3. **التطبيق يفتح فوراً!**

### الطريقة 3: OTA Update (للتحديثات)

بعد التثبيت الأول:
```bash
# انشر تحديث جديد
npx expo publish

# المستخدمون يستلمون التحديث تلقائياً
```
