# كيفية تشغيل سيرفر OTP 📱

## الطريقة السريعة ⚡

افتح PowerShell أو Terminal في مجلد `server` وشغل:

```bash
node quick-start.js
```

هذا السكريبت سيقوم بـ:
- ✅ إنشاء ملف `.env` تلقائياً
- ✅ توليد hash لكلمة المرور
- ✅ تشغيل السيرفر على المنفذ 3000

## الطريقة اليدوية 🔧

### 1. إنشاء hash لكلمة المرور

```bash
cd server
node create-hash.js
```

انسخ الناتج (يبدو مثل: `$2b$10$...`)

### 2. إنشاء ملف `.env`

أنشئ ملف `.env` في مجلد `server` وأضف:

```env
PORT=3000
ADMIN_USER=owner
ADMIN_HASH=PASTE_HASH_HERE
JWT_SECRET=trex_shop_secret_key_change_in_production_2024
```

### 3. تشغيل السيرفر

```bash
npm start
```

## التحقق من عمل السيرفر ✅

بعد تشغيل السيرفر، سترى في الـ console:

```
Auth server listening on 3000
```

## ملاحظات مهمة 📝

- ✅ السيرفر يعمل في وضع التطوير بدون Twilio
- ✅ رمز OTP سيظهر في console (لا حاجة لـ SMS حقيقي)
- ✅ العنوان: `http://localhost:3000`
- ✅ للعمل من الهاتف: استخدم IP جهازك بدلاً من localhost

## استكشاف الأخطاء 🔍

إذا واجهت مشكلة:
1. تأكد أن Node.js مثبت
2. تأكد أن جميع المكتبات مثبتة: `npm install`
3. تأكد أن المنفذ 3000 غير مستخدم
