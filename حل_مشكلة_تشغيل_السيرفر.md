# ✅ حل مشكلة تشغيل السيرفر - PowerShell Path

## ❌ **المشكلة:**

```powershell
cd server
cd : Cannot find path 'C:\Users\Abo Hamza\Desktop\T-REX_Shop_fu\server\server' 
because it does not exist.
```

و

```powershell
node index.js
Error: listen EADDRINUSE: address already in use 0.0.0.0:4001
```

---

## 🔍 **الأسباب:**

### 1️⃣ **أنت بالفعل في مجلد server!**

```powershell
PS C:\Users\Abo Hamza\Desktop\T-REX_Shop_fu\server> cd server
```

**الخطأ:** حاولت الدخول لـ `server` مرة أخرى بينما أنت بالفعل فيه!

**الحل:** لا تكتب `cd server` إذا كنت بالفعل في المجلد!

---

### 2️⃣ **السيرفر شغال من قبل!**

```powershell
Error: listen EADDRINUSE: address already in use 0.0.0.0:4001
```

**السبب:** هناك عملية node شغالة بالفعل على المنفذ 4001!

**الحل:** أوقف العملية القديمة أولاً!

---

## ✅ **الحل الصحيح:**

### الطريقة 1: إذا كنت خارج مجلد server

```powershell
# اذهب للمسار الكامل (استخدم علامات اقتباس)
cd 'C:\Users\Abo Hamza\Desktop\T-REX_Shop_fu\server'

# ثم شغّل السيرفر
node index.js
```

---

### الطريقة 2: إذا كنت داخل مجلد server

```powershell
# لا تكتب cd server مرة أخرى!
# فقط شغّل:
node index.js
```

---

### الطريقة 3: إعادة تشغيل كاملة (الأفضل)

```powershell
# 1. أوقف جميع عمليات node:
Get-Process -Name node -ErrorAction SilentlyContinue | Stop-Process -Force

# 2. انتظر ثانيتين:
Start-Sleep -Seconds 2

# 3. اذهب للمجلد الصحيح:
cd 'C:\Users\Abo Hamza\Desktop\T-REX_Shop_fu\server'

# 4. شغّل السيرفر:
node index.js
```

## 🎯 **التشخيص السريع:**

### تحقق من موقعك الحالي:

```powershell
# اعرف أين أنت:
pwd

# أو:
Get-Location
```

**النتيجة المتوقعة:**
```
Path
----
C:\Users\Abo Hamza\Desktop\T-REX_Shop_fu\server
```

---

### تحقق من العمليات الشغالة:

```powershell
# هل هناك node شغال؟
Get-Process -Name node -ErrorAction SilentlyContinue

# إذا ظهرت نتائج، أوقفها:
Get-Process -Name node | Stop-Process -Force
```

---

### تحقق من المنافذ:

```powershell
# هل المنفذ 4001 مشغول؟
netstat -ano | findstr :4001

# إذا نعم، أوقف العملية:
# 1. اعرف Process ID (PID) من العمود الأخير
# 2. Stop-Process -Id <PID> -Force
```
