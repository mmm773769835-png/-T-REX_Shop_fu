# حل مشاكل بناء تطبيق Android - T-REX Shop

## المشاكل الشائعة وحلولها

### 1. خطأ في رفع الملفات إلى EAS Build (ECONNRESET)
**السبب**: انقطاع في الاتصال بالإنترنت أثناء رفع ملفات المشروع
**الحل**: 
- تحقق من اتصال الإنترنت
- أعد المحاولة بعد اتصال أقوى
- حاول بناء المشروع في وقت مختلف

### 2. مشكلة في الاعتمادية
**السبب**: تعارض بين إصدارات الحزم
**الحل**: 
- استخدم `npm install --legacy-peer-deps`
- قم بتحديث الحزم المطلوبة

### 3. مشكلة في تكوين التطبيق
**السبب**: خصائص غير مدعومة في app.json
**الحل**: 
- تحقق من التكوين باستخدام `npx expo-doctor`
- قم بإزالة الخصائص غير المدعومة

## خطوات لحل مشكلة البناء

### الخطوة 1: التحقق من التكوينات
```bash
npx expo-doctor
```

### الخطوة 2: تنظيف ذاكرة التخزين المؤقت
```bash
npx expo prebuild --clean
npx eas build --clear-cache
```

### الخطوة 3: التأكد من اتصال قوي بالإنترنت
- قم بقياس سرعة الإنترنت
- أغلق التطبيقات الأخرى التي تستخدم الإنترنت

### الخطوة 4: إعادة محاولة البناء
```bash
npx eas build --platform android --profile preview
```

## بدائل لبناء التطبيق

### 1. بناء محلي (Local Build)
```bash
npx expo run:android
```

### 2. استخدام VPN
- إذا كانت هناك قيود على الشبكة في منطقتك
- استخدم اتصال VPN مستقر

### 3. البناء في أوقات مختلفة
- حاول البناء في أوقات قليلة الاستخدام للإنترنت
- من 12 صباحًا إلى 6 صباحًا غالبًا تكون أفضل

## معلومات فنية مهمة

- حجم ملفات المشروع: ~1.5 ميجابايت
- مدة البناء التقريبية: 15-45 دقيقة
- نوع الملف الناتج: APK (للاختبار) أو AAB (للمتجر)

## ملاحظات هامة

- تأكد من تسجيل الدخول إلى حساب Expo: `npx expo login`
- راقب صفحة البناء على موقع Expo: https://expo.dev/accounts/trexshop/projects/trex-shop
- سجلات البناء تحتوي على معلومات تفصيلية عن أي أخطاء
