## ✅ الخطوات التالية

### 1️⃣ تأكد أن السيرفر يعمل

السيرفر يجب أن يعمل في Terminal منفصل:
```

```

بعد التأكد من السيرفر:
1. اضغط `r` في Terminal لتحديث التطبيق
2. أو اضغط زر "RELOAD" في التطبيق

### 3️⃣ تحقق من Console

ستظهر رسالة في console:
```
🔗 API Server URL: http://10.56.255.204:4000
```

### 4️⃣ اختبر الاتصال

افتح المتصفح واذهب إلى:
```
http://10.56.255.204:4000/api/auth/check
```

يجب أن ترى رد من السيرفر.

## ⚠️ ملاحظات مهمة

1. **السيرفر يجب أن يعمل** في Terminal قبل تجربة OTP
2. **الهاتف والكمبيوتر** يجب أن يكونا على نفس شبكة WiFi
3. **Firewall** قد يحتاج لإذن - تأكد أن المنفذ 4000 مفتوح
